## 215. Kth Largest Element in an Array

```java
class Solution {
    public int findKthLargest(int[] nums, int k) {
        PriorityQueue<Integer> pQ = new PriorityQueue<>(Collections.reverseOrder());
        int answer = 0;

        for(int num : nums) {
            pQ.add(num);
        }

        for(int i = 0; i < k; i++) {
            answer = pQ.poll();
        }

        return answer;
    }
}
```
