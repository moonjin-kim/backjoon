1071. Greatest Common Divisor of Strings

단순 모든 경우의 수를 체크하여 결과 나타내기

```java
class Solution {
    public String gcdOfStrings(String str1, String str2) {
        int minLen = Math.min(str1.length(), str2.length());

        for(int i=minLen; i >= 1; i--) {
            if(str1.length()%i != 0 || str2.length()%i != 0) continue;
            boolean same = true;
            for(int j = 0; j < str1.length(); j++) {
                if(str1.charAt(j) != str1.charAt(j%i)) {
                    same = false;
                    break;
                }
            }

            for(int j = 0; j < str2.length(); j++) {
                if(str2.charAt(j) != str1.charAt(j%i)) {
                    same = false;
                    break;
                }
            }

            if(same) {
                return str1.substring(0,i);
            }
        }

        return "";
    }
}
```

두 배열을 합하였을때 앞으로 합하든 뒤로 합하든 결과가 같아야 최대 공약수를 구할 수 있다.

그러므로 글자의 길이의 최대 공약수가 곧 문자의 최대 공약수가 된다

```java
class Solution {
    public String gcdOfStrings(String str1, String str2) {
        if (!(str1 + str2).equals(str2 + str1)) {
            return "";
        }

        int lenGCD = gcd(str1.length(), str2.length());
        return str1.substring(0, lenGCD);
    }

    private int gcd(int len1, int len2) {
        while (len2 != 0) {
            int temp = len1 % len2;
            len1 = len2;
            len2 = temp;
        }
        return len1;
    }
}
```
