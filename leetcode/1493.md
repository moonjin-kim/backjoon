## 1493. Longest Subarray of 1's After Deleting One Element
```java
class Solution {
    public int longestSubarray(int[] nums) {
        int max = 0;
        int sum = 0;
        int start = 0;
        int end = 0;
        while(end < nums.length) {
            sum += nums[end++];
            if(end-start <= sum + 1) {
                max = Math.max(max, end-start-1);
            } else {
                sum -= nums[start];
                start++;
            }
        }

        return max;

        
    }
}
```
