## 643. Maximum Average Subarray I

```java
class Solution {
    public double findMaxAverage(int[] nums, int k) {
        int sum = 0;
        double max = 0.0;
        for(int i =0; i < k; i++) {
            sum += nums[i];
        }
        max = (double) sum / k;

        for(int i =1; i <= nums.length - k; i++) {
            sum -= nums[i-1];
            sum += nums[i + k - 1];
            max = Math.max(max, (double) sum / k);
        }

        return max;
    }
}
```
