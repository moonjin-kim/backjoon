# 724. 피벗 인덱스 찾기

```java
class Solution {
    public int maxOperations(int[] nums, int k) {
        int start = 0;
        int end = nums.length - 1;
        int answer = 0;

        Arrays.sort(nums);
        while(start < end) {
            int sN = nums[start];
            int eN = nums[end];

            if(sN + eN == k) {
                start++;
                end--;
                answer++;
            } else if(sN + eN > k) {
                end--;
            } else {
                start++;
            }
        }

        return answer;
    }
}
```
