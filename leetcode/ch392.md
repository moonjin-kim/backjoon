## 392. Is Subsequence

```java
class Solution {
    public boolean isSubsequence(String s, String t) {
        int start = 0;

        for(int i = 0; i < t.length(); i++) {
            if(start >= s.length()) {
                return true;
            }

            char sA = s.charAt(start);
            char tA = t.charAt(i);

            if(sA == tA) {
                start++;
            }
        }

        if(start >= s.length()) {
            return true;
        }

        return false;
    }
}
```
