## 343. Integer Break

```java
class Solution {
    public int integerBreak(int n) {
        int[] dp = new int[59];
        dp[1] = 1;
        
        for(int i = 2; i <= n; i++) {
            for(int j = 1; j <= i/2 + 1; i++) {
                dp[i] = Math.max(j * dp[i-j], dp[i]);
            }
            int half = i / 2;
            dp[i] = Math.max(dp[i], (1 - half) * half);
        }
        
        return dp[n];
    }
}
```
