## 799. Champagne Tower
```java
class Solution {
    public double champagneTower(int poured, int query_row, int query_glass) {
        double[][] champagne = new double[101][101];
        
        champagne[0][0] = poured;
        for(int i = 0; i < 100; i++) {
            for(int j = 0; j <= i; j++) {
                if(champagne[i][j] > 1) {
                    double low = (champagne[i][j] - 1) / 2;
                    champagne[i+1][j] += low;
                    champagne[i+1][j+1] += low;
                    champagne[i][j] = 1;
                }
            }
        }

        return champagne[query_row][query_glass];
    }
}
```
