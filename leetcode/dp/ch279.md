## ch279. Perfect Squares

```java
class Solution {
    public int numSquares(int n) {
        int[] dp = new int[n+1];
        for(int i=0; i < n+1; i++) {
            dp[i] = 1000000;
        }
        dp[0] = 0;
        int count = 1;
        // count가 n의 완전 제곱수 
        while(count*count <= n) {
            int sq = count * count;
            for(int i = sq; i < n + 1; i++) {
                dp[i] = Math.min(dp[i-sq] + 1, dp[i]);
            }
            count++;
        }
        return dp[n];
    }
}
```

`count * count`가 `n` 이하인 동안 반복문을 수행하여, `n` 이하의 모든 완전 제곱수를 하나씩 확인합니다.

각 완전 제곱수 `sq = count * count`에 대해,

`sq` 이상 `n` 이하의 모든 수 `i`에 대해 다음을 계산합니다.

- `dp[i - sq] + 1`

  → `(i - sq)`를 만드는 데 필요한 최소 완전 제곱수 개수에 현재 완전 제곱수 `sq` 하나를 추가한 경우

- 기존의 `dp[i]`

이 두 값 중 더 작은 값을 `dp[i]`에 저장합니다.

즉,

**각 수 `i`를 만들 수 있는 모든 완전 제곱수 조합 중에서, 사용되는 완전 제곱수의 최소 개수를 구하는 과정**입니다.

최종적으로 `dp[n]`에는 `n`을 만들 수 있는 완전 제곱수의 최소 개수가 저장됩니다.
