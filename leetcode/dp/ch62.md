## 62. Unique Paths

```java
class Solution {
    int[][] dic = {{-1,0}, {0,-1}};
    public int uniquePaths(int m, int n) {
        int[][] path = new int[m][n];
        path[0][0] = 1;

        for(int i=0; i<m; i++) {
            for(int j=0;j<n;j++) {
                for(int target=0; target<2; target++) {
                    int beforeY = i + dic[target][0];
                    int beforeX = j + dic[target][1];
                    if(beforeY >= 0 && beforeX >= 0) {
                        path[i][j] += path[beforeY][beforeX];
                    } 
                }
            }
        }

        return path[m-1][n-1];
    }
}
```
