## 650. 2 Keys Keyboard
```java
class Solution {
    public int minSteps(int n) {
        int[] dp = new int[n+1];
        
        for(int i = 2; i <= n; i++) {
            int min = i;
            for(int mod = 2; mod * mod <= i; mod++) {
                if(i%mod == 0) {
                    min = Math.min(dp[i/mod] + mod,min);
                }
            }

            dp[i] = min;
        }

        return dp[n];
    }
}
```
