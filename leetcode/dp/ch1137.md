## 1137. N-th Tribonacci Number

```java
class Solution {
    public int tribonacci(int n) {
        int[] tribo = new int[38];
        tribo[1] = 1;
        tribo[2] = 1;

        for(int i = 3; i <= n; i++) {
            tribo[i] = tribo[i-1] + tribo[i-2] + tribo[i-3];
        }

        return tribo[n];
    }
}
```
