## Letter Combinations of a Phone Number

```java
class Solution {
    String[] phone = {"", "", "abc","def","ghi","jkl","mno","pqrs","tuv","wxyz"};
    public List<String> letterCombinations(String digits) {
        List<String> answer = new ArrayList();

        Map<Character, String> digitToLetters = new HashMap<>();
        digitToLetters.put('2', "abc");
        digitToLetters.put('3', "def");
        digitToLetters.put('4', "ghi");
        digitToLetters.put('5', "jkl");
        digitToLetters.put('6', "mno");
        digitToLetters.put('7', "pqrs");
        digitToLetters.put('8', "tuv");
        digitToLetters.put('9', "wxyz");

        backtracking(digits, 0, "", answer, digitToLetters);

        return answer;
    }

    public void backtracking(String digits, int index, String cur, List<String> answer, Map<Character, String> digitToLetter){
        if(digits.length() == index) {
            answer.add(cur);
            return;
        }

        String letters = digitToLetter.get(digits.charAt(index));
        for (char data: letters.toCharArray()) {
            backtracking(digits, index+1, cur + data, answer,digitToLetter);
        }

        return;
    }
}
```
