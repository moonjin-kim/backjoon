## leetCode343.Integer Break

```java
class Solution {
    public int integerBreak(int n) {
        int[] dp = new int[59];
        dp[1] = 1;
        dp[2] = 1;

        for(int i = 3; i <= n; i++) {
            System.out.println(i + " (");
            for(int j = 1; j < i; j++) {
                int leftMax = Math.max(j, dp[j]);
                int rightMax = Math.max(i-j, dp[i-j]);
                int max = leftMax * rightMax;
                dp[i] = Math.max(dp[i],max);
            }
        }

        return dp[n];
    }
}
```
