## 정수 삼각형

```java
class Solution {
    public int solution(int[][] triangle) {
        int len = triangle.length;
        int[][] sum = new int[len][len];
        sum[0][0] = triangle[0][0];
        
        for(int i=0; i< len - 1; i++) {
            for(int j=0; j<=i; j++) {
                sum[i+1][j] = Math.max(
                    sum[i][j]+triangle[i+1][j],
                    sum[i+1][j]
                );
                sum[i+1][j+1] = Math.max(
                    sum[i][j]+triangle[i+1][j+1],
                    sum[i+1][j+1]
                );
            }
        }
        
        int answer = 0;
        
        for(int i=0; i < len; i++) {
            answer = Math.max(answer, sum[len-1][i]);
        }
        
        return answer;
    }
}
```
